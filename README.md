## wublog  创建一个博客项目用于学习记录自己遇到的知识点

 其中包含以下知识点
 * javascript
 * css
 * html
> 项目开发中

## 已完成的点

1.layout 布局通过命名路由实现
2.响应式适配rem
3.主题换肤通过var() css变量实现 (less穿透
4.代码高亮，支持多种语言代码
5.文章上传支持图片粘贴，文本转义


本项目前端使用vue-cli 2.x 开发
